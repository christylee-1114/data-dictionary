AMR data

Last update on 2024/09/18

100,000 rows of 12 variables

These data represent the sort of data that might be obtained from the 2nd generation surveillance system

* id Integer - A unique indentifier for each person 
* dob Character - a string giving the date of birth 
* spec_date Character - string giving the data a specimen was taken 
* sex_male Binary - indicatess whether the person from whom the specimen was taken was male or not. 1 (male) 0 (male)
